# Sujets produit — Abomey

Ce dossier contient les spécifications produit d'Abomey, sujet
par sujet. Chaque sujet est une tranche de produit cadrée,
formalisée et découpée en incréments livrables selon le
protocole `product-discovery`.

Le suivi de l'état d'avancement des sujets (ouvert, en cours,
clos) vit dans `suivi/abomey.md`.

## Sujets actifs

- [Création d'une Partie](creation-partie.md) — formulaire de
  création (nom + Mode de tarot + sélection de Joueurs), avec
  création de Joueurs à la volée via modale.

## Sujets livrés

- [Comptes utilisateurs et isolation des données](comptes-utilisateurs.md)
  — inscription publique via Google ou Apple, isolation totale
  des données par utilisateur, conformité RGPD minimale.
